Magdalenian Disk Classification

Deep Learning Project on Archaeological Image Analysis using CNN & Data Augmentation

Project Overview

This project explores the classification and stylistic analysis of Magdalenian perforated disks using deep learning. By blending and rotating archaeological disk images, we created an expanded dataset to train a Convolutional Neural Network (CNN) for identifying human-crafted engravings.

Data Augmentation Pipeline

  • Input Disks: Archaeological images with human and non-human engravings
  • Transformations: Rotation (90°, 180°, 270°), Horizontal/Vertical Flipping
  • Blending: Scatter-mask based image mixing to simulate real-world degradation and diversity
  • Patch Extraction: Splitting disks into 8x8 image patches for CNN input

CNN Architecture

  • Input: Augmented 8x8 disk image patches
  • Convolutional layers with ReLU activation
  • MaxPooling layers to downsample
  • Fully connected Dense layers
  • Softmax output layer: Human vs. Non-Human
  • 5-fold Cross Validation for robust evaluation

Results

  • Blending & flipping increased training data variety
  • Model trained with augmented data showed improved accuracy
  • CNN accurately classified human vs. non-human patterns on disks

Conclusion

This interdisciplinary project successfully combined archaeology and AI to analyze disk engravings. Data augmentation and CNNs provided scalable tools to explore human symbolism in ancient artifacts.
